* Default config: `anthropic_filemap.yaml`
* `clippybotbench_submissions`: Configs that were used for clippybotbench submissions
* `clippybot_0_7`: Configs from clippybot 0.7, similar to the one used in the paper
* `exotic`: Various specific configurations that might be more of niche interest
* `human`: Demo/debug configs that have the human type commands and run without a LM
* `demo`: Configs for demonstrations/talks
* Configs for running with SWE-smith are at https://github.com/clippybot-bench/SWE-smith/blob/main/agent/swesmith_infer.yaml
